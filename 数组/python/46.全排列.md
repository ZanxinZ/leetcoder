## 全排列

使用回溯 + 递归的思路。交换 1 和 其余各项，递归其余部分，交换的部分恢复。递归的终止条件是 i 与 j 相遇。


```python

class Solution:
    def permute(self, nums: List[int]) -> List[List[int]]:
        n = len(nums)
        self.nums = nums
        self.res = []
        self.doPermute(0, n - 1)
        return self.res
        
    def doPermute(self, i, j):
        if i == j:
            s = []
            for e in self.nums:
                s.append(e)
            self.res.append(s)
            return
        for index in range(i, j + 1):
            self.swap(i, index)
            self.doPermute(i + 1, j)
            self.swap(i, index)
            
    def swap(self, i, j):
        temp = self.nums[i]
        self.nums[i] = self.nums[j]
        self.nums[j] = temp

```

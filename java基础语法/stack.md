### Stack    栈

属于 java.util 包

| 方法                       | 描述                             |
| -------------------------- | -------------------------------- |
| boolean empty()            | 判断栈顶是否空，不会抛出异常     |
| Object peek()              | 查看栈顶，但不删除。可能抛出异常 |
| Object pop()               | 弹栈                             |
| Object push()              | 入栈                             |
| int search(Object element) | 查找对象的位置，以1为基数        |

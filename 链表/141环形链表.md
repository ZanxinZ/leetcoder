## 141环形链表

### 思路

1、快慢指针，快（2个结点/次），慢（1个结点/次）。

2、fast.next == slow 说明有回路,返回 true。

3、fast.next==null 或者 fast.next.next == null 都说明尾部没有回路，返回false。

### 代码

```java
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) {
 *         val = x;
 *         next = null;
 *     }
 * }
 */
public class Solution {
    public boolean hasCycle(ListNode head) {
            ListNode slow,fast;
            slow = head;
            fast = head;
            if(fast == null)return false;//防止空指针异常
            while(fast.next != null){
                if(fast.next.next == null)break;
                if(fast.next == slow)return true;
                slow = slow.next;
                fast = fast.next.next;
            }
            return false;
    }
}
```

### 拓展

- 如果要求环的长度

  需要在快慢指针相遇后，再跑到第二次相遇，两次相遇之间移动的次数即为环的长度。

- 代码

  ```java
  /**
   * Definition for singly-linked list.
   * class ListNode {
   *     int val;
   *     ListNode next;
   *     ListNode(int x) {
   *         val = x;
   *         next = null;
   *     }
   * }
   */
  public class Solution {
      public boolean hasCycle(ListNode head) {
              ListNode slow,fast;
              slow = head;
              fast = head;
              int count = 0;
              if(fast == null)return false;
              while(fast.next != null){
                  if(fast.next.next == null)break;
                  if(fast == slow){
                      //有环
                      slow = slow.next;
                      fast = fast.next.next;
                      while(fast != slow){
                          slow = slow.next;
                          fast = fast.next.next;
                          count++;
                      }
                      count++;
                      System.out.print(count);
                      return true;
                  }
                  slow = slow.next;
                  fast = fast.next.next;
                  count++;
              }
              return false;
      }
  }
  ```

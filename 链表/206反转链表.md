## 206  反转链表

![image-20210118010506638](206反转链表.png)

### 思路

1、前两个结点特殊处理，记录下剩下的部分的头，然后把头继续往一个一个拼接上，在拼接过程需要记录剩下部分。

### 代码

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode reverseList(ListNode head) {
        ListNode nextOne,nextNext,pre;
        
        if(head == null)return null;
        if(head.next == null)return head;
        
        
		//把第三个的头用nextNext把握住
        nextOne = head.next;
        nextNext = nextOne.next;
		//让第一个和第二个倒序
        nextOne.next = head;
        head.next = null;

        pre = nextOne;
        head = nextOne;
        
        //从第三个开始（nextNext在第三个），把它作为head,把它后面的继续把握住。现在只需把pre接到head 之后就可以。
        while(nextNext != null){
            head = nextNext;
            nextNext = head.next;

            head.next = pre;
            
            pre = head;
        }

        return head;
    }
}
```
